# Vault Search

Use this skill when the first job is to find where the answer lives.

## Workflow

1. Search the most likely hubs and reference notes.
2. Expand into nearby project or operational notes if needed.
3. Build a compact context pack from the best sources.
4. Highlight gaps, contradictions, and stale material.

## Output

- best matching notes
- short evidence summary
- suggested next query if context is still incomplete
